# Eng. Md. Amirul Islam — Personal Website

This is a simple static website built for Eng. Md. Amirul Islam.

To publish with GitHub Pages:
1. Create a repository named `yourusername.github.io`
2. Upload all files from this folder to the repository root
3. In Settings → Pages select the main branch as source

Replace `amirul@example.com` with your real email in index.html
